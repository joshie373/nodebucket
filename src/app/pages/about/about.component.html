Welcome to Joshua Hughes' Nodebucket!

nodebucket, is task management software designed to simplify the way you create, track, edit, and delete time sensitive material. This is a full-stack MEAN (MongoDB,Express, Angular, and Node.js) application, utilizing the concepts that were taught in previous courses. MongoDB will be used for data persistence. Node.js will be used for manipulating and returning saved records, SoapUI for unit testing, and Angular for user interactions.

How to use Nodebucket?

To start using Nodebucket, you will need to be logged in. You will need to click the Login button in the top-right of the page.

User Role:

To get to the nodebucket task manager, click on nodebucket in the menu


You will see you task management page here. These elements are draggable in between the todo, doing, and done columns.
To delete a task: click the red 'x' next to each task.


To add a new task: click the New Task button
once the Modal pops up, enter in the task's text
click "add task" to add the task, or "cancel" to cancel.

*all new tasks are automatically added to the to-do column.


Admin Role:

To get to the nodebucket admin homepage, click on nodebucket in the menu


You will see a list of options here.
To manage tasks: click the Manage Tasks button
To manage Users: click the Manage Users button
To view reports: click the View Reports button


Managing Tasks:

To add a task:Click the New task button

To edit a task:Click the edit button next to the task

To delete a task:Click the delete button next to the task


Managing Users:

To add a user:Click the New Employee button

To edit a user:Click the edit button next to the user

To delete a user:Click the delete button next to the user

*note: Core users cannot be deletes these are employee ids 1007,1008,1009,1010,1011,1012.

Viewing Reports:

Reports will show the totals for each user, as well as the stats for the team
